# Individul Apps

Monorepo pentru aplicațiile web personale.

## Aplicații

- **Portal** - Landing page cu split layout
- **Clasificare** - Calculator Fracțiuni (React + Vite + Tailwind)
- **PDF Toolbox** - Instrumente PDF (FastAPI + Docker)

## Deploy

Automat via GitHub Actions la fiecare push pe branch-ul `main`.

## Server

- Hetzner CX22
- Ubuntu 22.04
- Nginx + Docker
